# Physics Analyzer
A Python script that calculates force and energy for different objects using basic physics formulas.

## Run with Python 3
python3 physics-analyzer.py

## Example Output
Train: Force = 226800 N, Energy = 2041200000000000000000 J  
Car: Force = 4500 N, Energy = 135000000000000000000 J  
Boulder: Force = 4900.0 N, Energy = 45000000000000000000 J  

### Formulas Used

-Force: F = m × a  
(mass × acceleration)

-Energy: E = m × c²  
(mass × speed of light squared, where c = 3 × 10⁸ m/s)

### Concepts Practiced
-Integrating loops/functions/list manipulation   
-Default parameters in function def  
-Printing with f-strings and scientific notation
